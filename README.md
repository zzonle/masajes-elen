# 💅 Masajes Elen - Plataforma Web y Catálogo de Servicios

🔗 **Despliegue en Producción:** [https://masajes-elena.vercel.app](https://masajes-elena.vercel.app)

## 📌 Acerca del Proyecto

Desarrollo frontend a medida de una Single Page Application (SPA) para "Nails Elena", un negocio del sector de belleza especializado en manicura y nail art profesional. El objetivo principal de este proyecto fue establecer la presencia digital de la marca, digitalizar su catálogo de servicios y mejorar la captación de clientes mediante una interfaz moderna, rápida y orientada a la conversión.

## 🎯 Características Principales

* **Arquitectura SPA:** Navegación fluida y sin recargas de página utilizando Vue Router para una experiencia de usuario (UX) ininterrumpida.
* **Catálogo Dinámico de Servicios:** Módulo estructurado para la presentación detallada de tratamientos, descripciones y tarifas.
* **Portafolio Visual (Galería):** Sección interactiva optimizada para exhibir trabajos realizados, vital para la adquisición de clientes en el sector estético.
* **Módulo de Social Proof:** Integración de testimonios y reseñas para validar la calidad del servicio y generar confianza.
* **Diseño Mobile-First:** Interfaz de usuario (UI) completamente responsiva, garantizando una correcta visualización desde smartphones, tablets y desktop.
* **Performance Optimizado:** Tiempos de carga mínimos y gestión eficiente de *assets* gracias a la construcción con Vite.
